# Tempo v0.2.0 双人邀约内测版

## 版本说明

本版本重点完成双人“找时间”和邀约闭环，适合邀请朋友进行 Android 内测。

## 新增功能

- 支持好友搜索、好友申请、同意、拒绝和删除。
- 支持查看好友未来一周的红/绿/黄状态色块。
- 发起邀约时选择活动持续时间，支持 15 分钟至 8 小时。
- 支持在未来 7 天周表格中框选可约范围，例如“周三至周日每天 17:00—22:00”。
- 系统优先匹配双方完全空闲时间，没有结果时再纳入机动和未标记时间。
- 候选以完整的开始时间和结束时间展示，用户确认候选后发送邀约。
- 发起方可以取消邀约，接收方可以同意或拒绝邀约。
- 接收方同意后，已确认邀约会写入本地日历。
- 同一接收方存在重叠待应答邀约时，确认其中一个后其他邀约会自动拒绝。

## 使用方式

1. 使用两个账号登录，并先互加好友。
2. 在“找时间”中选择好友，点击“发起邀约”。
3. 输入活动名称，选择活动持续时间。
4. 可选：在未来 7 天周表格中框选日期和每天的可约时间范围。
5. 点击“扫描可用时间”，选择一个候选时间段并发送。
6. 使用接收方账号同意或拒绝邀约。

## 已知限制

- 当前仅支持 Android 内测。
- 群组多人撮合暂未完成。
- 正式推送通知暂未完成，当前主要通过应用内状态查看。
- 发起方灰色时间锁定、接收方改红自动失效等高级规则仍在开发。
- 当前版本建议使用测试账号，不要填写敏感个人信息。
