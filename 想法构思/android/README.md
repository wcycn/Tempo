# 互通日历 Android 原型

这是面向 Android 手机的 Jetpack Compose 原生原型，视觉参考了 PRD 中的黑底、深灰卡片、橙红强调色风格。

## 打开方式

用 Android Studio 打开 `想法构思/android`，等待 Gradle 同步后运行 `app` 模块即可。建议使用 Android Studio Jellyfish 或更新版本，JDK 17，Android SDK 35。

## 已实现的页面

- 日程：月历、状态图例、今日时间线、创建日程弹窗
- 找时间：待应答邀约、同意/换时间入口
- 群组：接龙人数、截止倒计时、成团进度
- 通知：邀约和群组动态
- 我的：健康提醒、外部同步、默认导入分类

## 后续接入顺序

1. 用 Room 保存日程、好友、邀约和群组数据。
2. 用 WorkManager 做外部日历同步与本地提醒。
3. 将匹配扫描抽成纯 Kotlin domain 层，覆盖“纯绿色扫描”和“绿色 + 黄色机动扫描”。
4. 接入登录、云端实时同步和推送通知。
5. 再补充文件导入、农历/节气和周/日拖拽网格。
